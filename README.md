# 自动上架脚本 — 三角洲行动

自动识别仓库物品价格，与军需回收价比较后决策是否上架，支持精准模式和框选批量模式。

---

## 使用前提

- Windows 10 / 11
- 分辨率与标定时保持一致（更换分辨率后需重新标定）

---

## 快速开始

1. 解压压缩包，双击 `lazydeal.exe` 运行（会弹出 UAC 管理员权限请求，必须同意）
2. 首次运行会提示坐标标定，按照引导完成标定后即可使用
3. 托盘图标右键可进行坐标重新标定、跟随游戏进程等设置

---

## 坐标标定说明

标定分为 4 个阶段，需要在游戏中依次打开对应界面：

| 阶段 | 需要打开的界面 |
|------|--------------|
| 阶段1 | 出售仓库物品弹出的选择面板 |
| 阶段2 | 点击上架按钮后弹出的上架面板 |
| 阶段3 | 关闭所有面板，回到仓库界面 |
| 阶段4 | 选择状态提示的显示位置 |

每个阶段按照提示将鼠标移到目标位置，按对应数字键记录，全部完成后按 ESC 进入下一阶段。

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `` ~+1+2 `` | 启用 / 禁用程序 |
| `左Shift + 左键` | 精准模式：点击物品触发上架 |
| `` ~+1 `` | 框选模式：记录起始点 |
| `` ~+2 `` | 框选模式：记录终点并开始（框选中再按强制停止） |

---

## 上架逻辑

1. 触发后自动弹出选择面板
2. OCR 识别军需回收价，同时检测物品是否可交易
3. 点击上架，进入上架面板
4. 点击第4根柱子获取快速出售价格，OCR 识别预期收入
5. **预期收入 > 军需回收价** → 上架；否则关闭面板跳过

---

## 跟随游戏进程

托盘图标右键勾选「跟随游戏进程」后选择游戏 exe，程序会：
- 游戏启动时自动启动脚本
- 游戏关闭时自动关闭脚本
- 注册开机自启，下次开机后自动生效

> 注意：移动程序文件夹后需重新勾选一次以更新路径。

## 文件说明

```
lazydeal/
├── lazydeal.exe      主程序
├── config.json       配置文件（首次运行自动生成，保存标定坐标）
├── logs/             运行日志（按日期自动生成）
```

---

## 注意事项

- 游戏需要以无边框窗口方式运行
- 程序需要管理员权限才能监听全局快捷键
- 标定坐标保存在 `config.json`，删除后需重新标定
- 更换显示器或调整游戏分辨率后需重新标定
- 请勿在标定或上架过程中移动游戏窗口
